How to Get Your Page

Your personal bio page will be hosted at:

https://bio.geekout.lol/<yourpage>

To claim your page:

1. Fork the official GitHub repository:
github.com/geekoutcoding/infinityfreebios

2. Inside your fork, create a file named domain.md.

3. Add the following required fields:

  • name — your real or preferred name
  • email — used only for verification
  • description — what your page should say
  • theme — dark, light, neon, minimal, etc.
  • domain — your page name (example: josh)

Example domain.md:

name: Josh
email: example@mail.com
description: Developer bio with links to GitHub + Discord.
theme: dark
domain: josh

What Happens Next

Once your request is reviewed and approved, your page will be generated and published.

You can request styles like:

  • Dark mode
  • Minimal white
  • Neon / cyber
  • Creator‑style layouts
  • Developer link hubs

Your final URL will look like:

https://bio.geekout.lol/<yourpage>
