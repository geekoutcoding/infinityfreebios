GeekOut

Bio Links

Claim your free, custom bio link when you build your website using the GeekOut Engine.

How It Works

1. Fork the official GitHub repository:
github.com/geekoutcoding/infinityfreebios

2. Create a file named domain.md inside your fork.

3. Inside that file, describe how you want your bio link to look using this format:

infree.hs.vc/<yourpage>

Example of a clean, professional bio link:

infree.hs.vc/josh

Or a more fun, creative one:

infree.hs.vc/pixelwizard

Or something simple and minimal:

infree.hs.vc/dev

Your free bio link format

Every page lives under: infree.hs.vc/<yourpage>

Just choose a unique page name (like josh, studio, or geekout) and we’ll host your bio at that URL.

How to Describe Your Bio Link

In your domain.md file, write a short description of how you want your link and page to appear. This helps us generate a clean, readable, and unique bio page for you.

You can describe things like:

  • Page name — what you want after infree.hs.vc/.
  • Theme — dark, neon, minimal, retro, glossy, etc.
  • Style — professional, playful, techy, aesthetic.
  • Purpose — portfolio, socials hub, project showcase, creator page.

Example description:

I want my bio link to be infree.hs.vc/josh.
Make it dark mode with neon blue accents.
Include links to my GitHub, Discord, and main GeekOut projects.
Keep it clean and minimal.
