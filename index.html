How to Get Your Bio Link

Your bio page will be hosted at:

https://infree.hs.vc/<yourpage>

To claim your page:

1. Fork the official GitHub repository:
github.com/geekoutcoding/infinityfreebios

2. Create a file named domain.md in your fork.

3. Add the following required fields:

  • name — your real or preferred name
  • email — used only for verification
  • description — what your page should say
  • theme — dark, light, neon, minimal, etc.
  • domain — your page name (example: infree.hs.vc/josh)

Example domain.md:

name: Josh
email: example@mail.com
description: Developer bio with links to GitHub + Discord.
theme: dark
domain: josh

What Your Page Will Look Like

Your page will be generated automatically based on your description and theme. You can request styles like:

  • Dark mode
  • Minimal white
  • Neon / cyber
  • Creator‑style layouts
  • Developer link hubs

Once approved, your page will go live at:

https://infree.hs.vc/<yourpage>
